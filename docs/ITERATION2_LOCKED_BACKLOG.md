# GEO V1 迭代2锁定清单（不遗漏）

以下三项为已锁定迭代能力，按优先级执行，后续开发不得跳过：

## 1. 内容生成专家智能体团队（最高优先级）

- 目标：把“策略师 -> 编辑 -> 审校 -> 平台改写”做成可配置多角色协同链路。
- 当前预埋：`PromptProfile` + 提示词装配器入口已就位。
- 迭代交付：多角色提示词串联、角色输出衔接、质量评估指标（结构完整度、可引用率、合规率）。

## 2. 公众号文章一键图文生成

- 目标：输入主题后，生成公众号图文所需正文、配图指令、排版导出。
- 当前预埋：前端按钮位 + `POST /api/v1/creative/wechat-rich-post` 占位接口。
- 迭代交付：正文+配图生产、图文导出（至少 html/md）、可选发布链路对接。

## 3. 专业 Skill 子步骤编排深度化

- 目标：把 skill 能力封装为可编排的流水线节点，支持重试、状态追踪、结果回传。
- 当前预埋：`WorkflowStep` 数据模型 + mock run 入口。
- 迭代交付：真实 adapter 接入（按平台/任务类型），失败重试策略，步骤级指标面板。

## 默认顺序

1. 专家提示词团队  
2. 公众号一键图文  
3. Skill 编排深度化
